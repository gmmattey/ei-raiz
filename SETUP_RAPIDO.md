# ⚡ Setup Rápido - 2 Minutos

## 🚀 OPÇÃO 1: Windows

1. Baixe e instale Node.js: https://nodejs.org/ (v18+)
2. Extraia/Clone o projeto: `Esquilo Invest`
3. **Clique duplo em:** `start-dev.bat`
4. Abra: http://localhost:3000 ✅

## 🚀 OPÇÃO 2: Mac/Linux

```bash
# 1. Instale Node.js (se não tiver)
# macOS: brew install node
# Linux: sudo apt-get install nodejs npm

# 2. Clone/extraia o projeto
cd "Esquilo Invest"

# 3. Inicie
chmod +x start-dev.sh
./start-dev.sh

# 4. Abra: http://localhost:3000 ✅
```

## 🚀 OPÇÃO 3: Docker (Qualquer SO)

```bash
# 1. Instale Docker Desktop: https://www.docker.com/products/docker-desktop
# 2. Inicie Docker Desktop
# 3. No terminal:
docker-compose up --build
# 4. Abra: http://localhost:3000 ✅
```

---

## ❓ Dúvidas?

- **Porta 3000 ocupada?** → Ver `AMBIENTE_LOCAL.md`
- **Erro "Node not found"?** → Instale Node.js de novo e reinicie o terminal
- **Quer acessar de outro PC?** → Ver `AMBIENTE_LOCAL.md` (seção "Acessar de outro computador")
