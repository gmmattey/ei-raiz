# Ícones Oficiais - Esquilo Invest

Esta pasta contém os ícones oficiais do projeto organizados por variações de cores da marca.

## Estrutura de Pastas

-   `original/`: Contém os arquivos SVG originais, com as propriedades de cores nativas (geralmente `currentColor`).
-   `preto/`: Ícones aplicados na cor **Deep Graphite** (`#0B1218`).
-   `laranja/`: Ícones aplicados na cor **Esquilo Orange** (`#F56A2A`).
-   `branco/`: Ícones aplicados na cor **Branco Puro** (`#FFFFFF`).

## Especificações Técnicas

-   **Preto (Deep Graphite):** `#0B1218`
-   **Laranja (Esquilo Orange):** `#F56A2A`
-   **Branco:** `#FFFFFF`

## Regras de Manutenção

1.  Os arquivos originais em `original/` devem ser preservados e nunca editados diretamente.
2.  Para adicionar um novo ícone, coloque o arquivo em `original/` e gere as variações de cores correspondentes.
3.  A geometria, traço e proporção (`viewBox`) são idênticos em todas as variações.
